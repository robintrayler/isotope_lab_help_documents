# Steps
1. From the Oracle Cloud main screen select `Purchase Requisitions` (@fig:fig1). 
2. From the requisitions page select `more tasks` and choose `Update Requisition Preferences` from the dropdown menu (@fig:fig2). 
3. Click the search button next to the field labeled `Deliver-to Location` (@fig:fig3). 
4. Click the `Advanced` button (@fig:fig4).
5. Under the `name` and `code` fields select `is not blank` and click `Search` (@fig:fig5). 
6. From the search results click on the row for the location you want to be your default. For deliveries to the Main UC Merced campus select `UC Merced Central Receiving` (@fig:fig6). Click `OK` to close the search results window and `Save and Close` to close the Edit Requisition Preferences window. The default Search location should now be set.
